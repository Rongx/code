# 编程规范
-------------

> 这是自己写的编程规范，所涉及的知识都是自己会的或是自己能够理解的，所以内容肯定是不够全面的。本着开源的精神，分享给大家作参考。

* [Android编程规范](android.md)

注意：

* 所有编程规范中的示例代码并不作为规范。有时候为了示例简洁，示例未必完全遵守编程规范（比如在变量命名等方面）。示例只是为了方便读者理解当前正在讲解的规范。
* 所有示例，如果没有说明，都是正确的示例。错误的示例会用 `// bad` 类似的注释说明。
